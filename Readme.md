- Fazer deploy de VueJS em Github Pages
1. `npm install --save-dev push-dir`
2. `npm run build`
3. Add `"deploy": "push-dir --dir=dist --branch=gh-pages --cleanup"` em `packages.json` dentro de `scripts`
